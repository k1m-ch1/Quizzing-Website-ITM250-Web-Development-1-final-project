PNG vs. JPG quiz

Information

The PNG and JPG file format are the most common file formats for storing images. PNG differ from JPG as it's a lossless compression file format, and has an extra alpha channel for opacity. Although JPG's encoding is lossy (there are many different forms of JPG, but the most common ones tend to be lossy), the image quality doesn't get impacted that much. This quiz aims to test whether a person can even tell the difference between PNG image and one that has been encoded using JPG.
