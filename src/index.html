Want to feature a quiz on our site?

We welcome community-submitted quizzes. Quizzes follow a simple HTML format — each question is a <section> with a <fieldset> for the choices, and the correct answer always goes first (we shuffle at load time so players won't notice).

To submit a quiz, reach out to us with your quiz file. We'll review it and add it to the site.

Format at a glance

  1. An <h1> for the quiz title
  2. A <section> for instructions / quiz info
  3. One <section> per question, each containing a <fieldset> with radio <input> options — correct answer listed first
